# GlowCare - Skin Care Recommendation System

A console-based skincare recommendation application developed in C++.

## Features
- User Registration
- User Login
- Personalized Skincare Recommendations
- Dietary Suggestions
- File Handling for Data Storage

## Technologies Used
- C++
- Object-Oriented Programming (OOP)
- File Handling

## Concepts Applied
- Classes
- Encapsulation
- Inheritance
